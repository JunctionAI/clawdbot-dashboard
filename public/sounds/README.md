# Notification Sounds

Place notification sound files here:

- `notification.mp3` - Default notification sound (recommended: short, subtle chime)
- `urgent.mp3` - Urgent notification sound (optional)

Recommended specifications:
- Format: MP3 or WAV
- Duration: 0.5-2 seconds
- Sample rate: 44.1kHz
- Bit rate: 128kbps (MP3)

Free sound sources:
- https://freesound.org
- https://mixkit.co/free-sound-effects
- https://www.zapsplat.com
